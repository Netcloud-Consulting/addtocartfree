== Description ==
Customize the 'Add to Cart' button text globally or per product with a lightweight and user-friendly UI.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/custom-add-to-cart-text` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Add to Cart Text to customize the button text.
